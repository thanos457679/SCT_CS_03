Password Strength Checker
Description
This is a simple Python-based Password Strength Checker that evaluates the strength of a password based on the following criteria:

Minimum 8 characters long
Atleast one uppercase letter
Atleast one lowercase letter
Atleast one number
Atleast one special character
The program also displays the missing requirements and classifies the password as Weak, Medium, or Strong.

Features
Checks password length
Checks uppercase letters
Checks lowercase letters
Checks numbers
Checks special characters
Displays password strength
Handles empty password input
Technologies Used
Python 3
Regular Expressions (re module)
